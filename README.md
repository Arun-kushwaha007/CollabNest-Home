## 🎯 **What is CollabNest?**

**CollabNest** is a next-generation collaborative workspace platform that revolutionizes how distributed teams work together. This landing page serves as the beautiful gateway to showcase the platform's capabilities.

### **🔗 Platform Overview**
- 🌍 **Global Collaboration:** Breaking geographical barriers for teams worldwide
- ⚡ **Real-time Synchronization:** Instant updates with Socket.io and Redis caching
- 🔒 **Enterprise Security:** OAuth 2.0, JWT tokens, and role-based access control
- 📱 **Cross-platform Access:** Seamless experience across web, mobile, and desktop
- 🤖 **Smart Features:** AI-powered task prioritization and intelligent notifications

## 🚀 **Quick Start**

### **📋 Prerequisites**

Before you begin, ensure you have:

| 🛠️ **Tool** | 📋 **Version** | 📥 **Download** |
|-------------|---------------|----------------|
| ![Node.js](https://img.shields.io/badge/Node.js-v18+-339933?style=flat-square&logo=node.js&logoColor=white) | 18.0.0+ | [Download](https://nodejs.org/) |
| ![npm](https://img.shields.io/badge/npm-v9+-CB3837?style=flat-square&logo=npm&logoColor=white) | 9.0.0+ | Included with Node.js |
| ![Git](https://img.shields.io/badge/Git-Latest-F05032?style=flat-square&logo=git&logoColor=white) | Latest | [Download](https://git-scm.com/) |

### **⚡ Installation Steps**

1. **📥 Clone the Repository**
   ```bash
   git clone https://github.com/Arun-kushwaha007/CollabNest-Home.git
   cd CollabNest-Home
   ```

2. **📦 Install Dependencies**
   ```bash
   npm install
   ```

3. **🚀 Start Development Server**
   ```bash
   npm run dev
   ```

4. **🌐 Open in Browser**
   ```
   http://localhost:5173
   ```

### **🏗️ Build for Production**

```bash
# Create optimized production build
npm run build

# Preview production build locally
npm run preview
```

## 🌓 **Theme System**

The landing page features a sophisticated dark/light mode system:

### **🎨 Theme Features**
- 🔄 **Automatic Detection** — Respects system preferences
- 💾 **Persistent Storage** — Remembers user choice via localStorage  
- ⚡ **Instant Switching** — Smooth transitions between themes
- 🎯 **CSS Variables** — Dynamic color system for consistent theming
- 📱 **Mobile Optimized** — Touch-friendly theme toggle

### **🔧 Theme Implementation**
```javascript
// Custom hook usage
const { isDark, toggleTheme } = useDarkMode();

// CSS variables automatically update
:root {
  --background: theme('colors.white');
  --foreground: theme('colors.gray.900');
  --primary: theme('colors.blue.600');
}

[data-theme="dark"] {
  --background: theme('colors.gray.900');
  --foreground: theme('colors.white');
  --primary: theme('colors.blue.400');
}
```
